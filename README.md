# instget
python  
インスタグラムのスクレイピング  

# 使い方
chromedriverの場所をフルパスで設定  
自分のインスタIDとパスワードを直書き  
inst_listにほしい人のインスタIDをリストで書く  
run.bat(python run.py)を実行
